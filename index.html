Java to Native

The RoboVM compiler compiles Java bytecode into native ARM or x86 code. Apps run directly on the CPU. No interpreter or virtual machine involved.

Read more »

Objective-C Bridge

RoboVM includes an Objective-C to Java bridge. Bridged Objective-C objects can be used just like any other Java object.

Read more »

Share Code

Easily share code between Android and iOS apps. The standard classes (java.lang.*, java.util.*, etc) included in RoboVM are based on Android's runtime.
